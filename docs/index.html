  • alio.go.kr 접속 → 기관 검색
  • 공시항목 페이지 하나씩 열기
  • 파일 하나씩 클릭 다운로드
  • 기관별 폴더 수동 정리
  • 355기관 × 92항목 = 반복 지옥
  • 1기관 1항목 ≈ 5분 → 총 2,722시간
→

  • 첨부파일 자동 분류·저장
  • 항목/기관별 폴더 자동 생성
  • 진행률·로그 실시간 표시
  • 355기관 전체 수집 ≈ 수 분

92개 전 항목 지원

ALIO 공시 92개 항목을 모두 수집할 수 있습니다. 항목 메타데이터는 자동 캐시됩니다.

⚡

병렬 다운로드

ThreadPoolExecutor 기반 병렬 처리로 기관당 5배 빠른 수집 속도를 제공합니다.

🎯

다중 필터

기관유형 · 주무부처 · 지역 · 키워드 검색을 조합해 원하는 기관만 골라냅니다.

☑

다중 선택

항목도, 기관도 복수 선택. 체크박스로 원하는 조합만 골라서 수집합니다.

📁

자동 폴더 정리

항목명 / 기관명 2단계 폴더 구조로 파일을 자동 분류·저장합니다.

엑셀 내보내기

INSTALL
설치 및 실행

Python 3.10+ 설치

이미 설치되어 있다면 건너뛰세요.

필수 패키지 설치

pip install requests openpyxl beautifulsoup4

프로그램 다운로드

alio_crawler_v5.4.py 와 alio_core.py 두 파일을 같은 폴더에 저장합니다.

실행

python alio_crawler_v5.4.py — GUI 창이 열리면 바로 사용 가능합니다.
